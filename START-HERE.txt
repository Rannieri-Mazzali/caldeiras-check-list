📊 API ENDPOINTS
═════════════════════════════════════════════════════════════════════════════

AUTENTICAÇÃO:
POST /api/auth/login
  → Body: { nickname, password }
  → Response: { success, token, nickname }

CHECK-LIST:
POST   /api/checklist/create
GET    /api/checklist/:id
PUT    /api/checklist/:id
POST   /api/checklist/:id/finalize

FOTOS:
POST   /api/photo/upload
  → FormData: { photo: File }
  → Response: { success, message, fotoPath }

MANUTENÇÃO:
POST   /api/maintenance/register
  → Body: { placa, km }
GET    /api/maintenance/check/:placa/:km
  → Response: { necessitaManutencao, kmRestantes, proximaManutencao }

❓ TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

❌ "Porta 3000 já está em uso"
   → npm run dev -- --port 3001

❌ "Câmera não funciona"
   → Use HTTPS em produção
   → Localhost funciona em desenvolvimento
   → Verifique permissões do navegador

❌ "Node não encontrado"
   → Instale: https://nodejs.org
   → Reinicie terminal
   → Digite: node --version

❌ "npm install não funciona"
   → rm -rf node_modules package-lock.json
   → npm cache clean --force
   → npm install

❌ "Fotos não são enviadas para WhatsApp"
   → Verifique número em .env
   → Use WhatsApp Web ou app instalado
   → Verifique conexão de internet

❌ "TypeScript não compila"
   → npm install
   → npm run build
